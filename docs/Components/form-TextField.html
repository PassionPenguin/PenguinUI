TextField

The TextField component enables a user to type text into an app. It's typically used to capture a single line of text, and the text displays on the screen in a simple, uniform format. We'd given them some special design base on Google's design.

Simple TextField
new TextField({placeholder: "City Name..", name: "City"})
With Color Styles
new TextField({placeholder: "City Name..", name: "City", color: "danger"})
Textarea
new TextField({placeholder: "Fear is nothing but the unknown things in your heart...", name: "Tips", size: "textarea"})
Separated Placeholder & Description
new TextField({placeholder: "Tianhe Park, Tianhe, Guangzhou, Guangdong...", description: "Address..", name: "Address"k})
With Prefix Icon
new TextField({placeholder: "Nationality..", name: "Nationality", prependicon: "flag"})
With Suffix Icon
new TextField({placeholder: "Language..", name: "Language", appendicon: "translate"})
